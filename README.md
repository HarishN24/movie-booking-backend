# 🎬 Movie Ticket Booking System – Backend

A production-ready backend application developed using Spring Boot, Spring Security, JWT Authentication, and MySQL. The system provides secure REST APIs for movie management, seat booking, user authentication, and booking history.

## 🚀 Live Backend

https://movie-booking-backend-production-934e.up.railway.app

## 📌 Project Overview

This application simulates a real-world movie ticket booking platform where users can:

* Register and Login securely
* Browse available movies
* Select seats and book tickets
* View booking history
* Prevent duplicate seat reservations
* Access protected APIs using JWT Authentication

The backend is deployed on Railway and connected to a cloud-hosted MySQL database.
